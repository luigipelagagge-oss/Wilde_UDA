2. Concetti Chiave & Esempi

A. Separation of Concerns

Principio per cui separiamo la struttura (HTML) dalla logica (JS).

HTML: <div id="app"></div> (Contenitore statico)
JS: renderApp() (Riempie il contenitore coi dati)
B. Event Listener (Ascolto Eventi)

Il codice che "ascolta" le azioni dell'utente (click, tastiera).

input.addEventListener('keyup', (e) => { ... });
C. Reazione DOM (Ciclo)

Input Utente -> Pulizia Schermo -> Filtro Dati -> Disegno Nuovi Dati.
